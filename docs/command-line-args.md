### Command line arguments
| Argument      | Description                   | default value     |
|---------------|-------------------------------|-------------------|
| -p,--port     | Rest client port              |          8089     |
| -r,--rest     | Enable rest client            |          true     |
| -h,--help     | Telegram spy help information |          false    |
| -v,--version  | Telegram spy version info     |          false    |
